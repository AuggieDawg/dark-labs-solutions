# Dark Labs Platform — Product Definition

## Working Definition

Dark Labs Platform is a private operating platform for managing the Dark Labs business and the founder's execution system.

It combines:

- a public business website
- authenticated client portal
- owner-only command center
- client and project CRM
- task workbench
- personal and business goal tracker
- knowledge base
- future automation and intelligence layer

## Product Areas

### Public Site

The public-facing Dark Labs business website. Its purpose is positioning, trust, lead generation, and proof of capability.

### Client Portal

Authenticated client-facing area where clients can view project status, milestones, deliverables, shared notes, documents, and updates.

### Owner Command Center

Owner-only operating area for managing clients, projects, tasks, goals, notes, metrics, and strategic execution.

### Workbench

Execution system for tasks, project dependencies, planning, relationship mapping, and weekly progress.

### Knowledge Layer

Long-term memory system for client notes, project decisions, technical notes, procedures, prompts, lessons learned, and reusable playbooks.

## Core Principle

The platform is built as a workspace-based operating platform, not a pile of disconnected pages.

Every protected object must be scoped by workspace, client, project, or explicit visibility.
