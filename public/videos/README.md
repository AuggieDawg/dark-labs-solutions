# Dark Labs Video Assets

Place purchased video assets here during local development.

Expected filenames used by the current UI:

- dark-labs-hero.mp4
- services-websites.mp4
- services-automations.mp4
- services-dashboards.mp4

Recommended specs:

- MP4 / H.264
- 1920x1080 minimum
- 10-25 second seamless loop
- dark cinematic footage
- no visible third-party logos
- no readable proprietary code
- keep file sizes controlled for web delivery

Later, production assets can move to object storage or a CDN.
